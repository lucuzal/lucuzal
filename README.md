## 📂 Projects

### 📚 Academic Indexing System (R)

A lightweight system designed to automate the creation of academic indexes following *The Chicago Manual of Style*.

- Transforms structured Excel tables (Entries & Subentries)  
- Outputs properly formatted Word documents  
- Designed for editorial workflows and academic publishing  

**Key features:**
- Data transformation and hierarchical structuring  
- Automated formatting rules  
- Reproducible workflow in R

### 🗄️ El Porteño Archive Manager (Python + GTK + MySQL)

A desktop application for exploring and editing a structured archive of the Argentine cultural magazine *El Porteño* (1982–1993).

- Built with **Python and GTK** for GUI interaction  
- Connected to a **MySQL database**  
- Designed for research and archival work  

**Key features:**
- Visual interface for browsing and editing records  
- Database integration and query handling  
- Focus on usability for humanities research

Un sistema liviano diseñado para automatizar la creación de índices académicos siguiendo las normas del *Chicago Manual of Style*.

- Orientada a investigación y trabajo archivístico  

**Características principales:**
